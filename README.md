# StaySense

StaySense is a web app hotel discovery and travel recommendation application built with MERN stack and full integration of AI. It helps travelers explore hotels, compare stay details, describe preferences in natural language, and receive personalized recommendations based on destination, also chat with AI to reach your best hotel for your trip.

## Features

- Hotel exploration with search, city, feature, price, and sort filters
- Hotel detail pages with gallery interaction and stay information
- **AI assistant experience for interpreting natural-language preferences**
- Personalized recommendations based on extracted travel preferences
- Review submission and hotel-specific review summaries
- Booking and checkout interface with reservation summary UI
- User profile with account details, reviews, and reservation history

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
